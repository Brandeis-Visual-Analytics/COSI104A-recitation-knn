# COSI104A-recitation-knn
Simple kNN exercise in a jupyter notebook for Brandeis University COSI 104A Intro to ML course

Please download this notebook and run it on your own computer during recitation.

If you are comfortable with github, you can clone this repository and work from your own local copy.

If you aren't comfortable with github, you can simply click the green Code button on the top right of the page, and then click Download Zip to download the whole repository.  After unzipping, the notebook should be found in the file `Recitation1-knn-COSI104a.ipynb`.
